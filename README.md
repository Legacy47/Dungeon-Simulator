# Dungeon-Simulator
* Python Text-Based Game

# Features
* Health
* Gold
* Items (Weapons, Shields, Armour etc..)
* Item Shop
* Fantasy (Goblins, Ogres, Witches, Dragonrace*** etc..) 
* Classes (Warrior, Archer, Summoner)
* Interface
* Main Menu
* Music
* Inventory
* Dialog Speed Interchangable
